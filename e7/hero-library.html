Fribbels E7 Hero Library

This app shows top builds globally per hero. Values of atk/hp/def are randomly modified by up to 2% for anonymity. Definitions of each column can be found on the optimizer github page.

Build score (bs) is an experimental measure for stat efficiency that accounts for main stat gear score and removes gs from overcapped crit.

Check out the Fribbels discord server for questions or more info! https://discord.gg/rDmB4Un7qg
